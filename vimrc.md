> insert in .vimrc file

```bash
set nocompatible
filetype off
filetype plugin indent on
set title
set noswapfile
set noshowmode
set noshowcmd
set sts=2 ts=2 st=2 sw=2 et ai si
set backspace=indent,eol,start
set bg=dark
set expandtab
set ruler
set ignorecase
set hlsearch
set number
set undofile
set termguicolors
syntax on
```
